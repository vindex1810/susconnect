# SUS Connect - Sistema de Monitoramento da Saúde

Sistema de monitoramento da saúde para Dourados/MS, desenvolvido para a Prefeitura Municipal.

## 🚀 Funcionalidades

- Dashboard com indicadores de saúde
- Relatórios de diabéticos
- Monitoramento de equipes de saúde
- Interface responsiva e moderna

## 🛠️ Tecnologias

- React + TypeScript
- Tailwind CSS
- Vite
- Lucide React (ícones)
